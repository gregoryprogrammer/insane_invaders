Insane Invaders
===============

Game written in ActionScript3 inspired by Space Invaders.
You can compile it with *mxmlc* from Flex SDK 4.6.

# Library: [Flixel 2.35][flixel].
# Music: RetroFuture Clean by [Kevin MacLeod][kevin].
